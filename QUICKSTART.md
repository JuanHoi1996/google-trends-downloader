# 快速开始 ⚡

5 分钟上手 Google Trends Downloader！

## 🎯 三步开始

### 1️⃣ 安装依赖（仅首次）

```bash
pip install -r requirements.txt
```

### 2️⃣ 运行程序

```bash
python google_trends_gui_no_warmup.py
```

### 3️⃣ 开始下载

1. 查看默认关键词（左侧）
2. 点击"开始下载"
3. 等待完成（约 5-8 分钟）
4. 查看 CSV 文件

**就这么简单！** ✨

## ⚠️ 常见问题

### ❌ 下载失败（429 错误）

**解决方法**：
1. 增加等待时间到 40-80 秒
2. 减少关键词数量
3. 稍后再试

### ❌ SSL 连接错误

**解决方法**：
1. 检查网络连接
2. 尝试使用 VPN
3. 重启程序

### ❌ 没有安装 tkinter

**解决方法**（Windows）：
```bash
# 重新安装 Python，勾选 tcl/tk
```

**解决方法**（Linux）：
```bash
sudo apt-get install python3-tk
```

## 💡 使用技巧

### 技巧 1：批量下载

如果有 20+ 个关键词：
- 分成 2-3 批，每批 8-10 个
- 每批之间等待 10 分钟

### 技巧 2：提高成功率

- 增加等待时间到 60-90 秒
- 在非高峰时段下载（凌晨/清晨）
- 使用稳定的网络连接

### 技巧 3：数据分析

用 Python 分析下载的数据：

```python
import pandas as pd
import matplotlib.pyplot as plt

# 读取数据
df = pd.read_csv('google_trends_20241124_143025.csv', index_col=0)

# 绘制趋势图
df.plot(figsize=(12, 6))
plt.title('Google Trends - 搜索热度')
plt.ylabel('相对热度')
plt.show()
```
